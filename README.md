# razel

命令行工具集合

## 安装

```bash
npm install -g razel
```

## 执行

1.同步源码映射地址
```bash
razel syncSourceMapUrl
```


## 帮助

```sh
$ razel --help
  Usage: razel [options] [name]
  
  Options:
    -v, --version        显示版本号
    -p, --port <number>  设置开发端口 (default: "8080")
    -d, --dir <string>   设置源码目录 (default: "dist-theme")
    -h, --help           display help for command
```
